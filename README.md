# Algeria-SocioEconomic-Trends (In Progress)

## Project Description
Analysis of socio-economic trends in Algeria.
Data Source: [Algeria Statistics 1960-2022](https://www.kaggle.com/datasets/bensalem14/algeria-statistics-1960-2022/data)

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run notebooks in `notebooks/` folder.
3. Configure project settings in `config.yaml`.
4. Run tests with `pytest`.
